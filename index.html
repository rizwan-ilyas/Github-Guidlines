Hi, Welcome here for Github Learning

  • to be get started first download and install the git by click here
  • to get status run the command "git status" in terminal
  • to add a file run the command "git add <filename>"
  • to commit a file run the command "git commit -m 'message'"
  • to clone a projet run the command "git clone <repository link>"
  • to link association with the repository run the command
    "git remote add origin "
  • to create a new branch run the command "git checkout <branch name>"
